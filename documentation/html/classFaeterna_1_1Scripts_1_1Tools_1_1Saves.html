Detailed Description

Clase centralizada para gestionar el guardado y carga de datos del juego, incluyendo partidas guardadas (slots) e imágenes de vista previa.

Maneja la persistencia de datos de juego utilizando archivos JSON y genera capturas de pantalla como previsualizaciones. También gestiona opciones de usuario como configuración de audio y video localStorage. Proporciona métodos para guardar, cargar, eliminar y verificar la existencia de partidas guardadas en diferentes slots.

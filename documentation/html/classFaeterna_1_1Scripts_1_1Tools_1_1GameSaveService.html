◆ PendingGameData

GameData Faeterna.Scripts.Tools.GameSaveService.PendingGameData
staticgetprivate set

Datos de juego pendientes de aplicar tras cargar/recargar una escena.

Se usan para almacenar temporalmente el GameData mientras se realiza ChangeScene y poder aplicarlo al nuevo nodo jugador cuando la escena se haya cargado.

Detailed Description

Gestor del cursor personalizado del juego.

Carga una textura de cursor personalizada desde el sistema de archivos, la redimensiona a un tamaño específico, y la aplica al juego como el cursor del mouse. Cuando el nodo se libera, restaura el cursor predeterminado del sistema.

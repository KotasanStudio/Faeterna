Detailed Description

Controlador del AnimationTree de Lira.

SETUP EN GODOT:

  1. Añade un nodo AnimationTree como hijo de Lira2D.
  2. En el inspector:
    • Tree Root → AnimationNodeStateMachine
    • Anim Player → ../AnimationPlayer
    • Active → true
  3. Crea los estados en el StateMachine: idle | run | jump | fall | dash | aim | shot | kick | getHit
  4. Añade este script al nodo AnimationTree.

TRANSICIONES RECOMENDADAS EN EL EDITOR: idle ──► run (auto, condición: blend_position != 0) run ──► idle (auto)

  • ──► jump (auto, se controla desde código) jump ──► fall (auto) fall ──► idle/run (auto, al aterrizar)
  • ──► dash (auto) dash ──► idle/run (auto) idle ──► aim (auto) run ──► aim (auto) aim ──► idle (auto)
  • ──► kick (OneShot encima del estado base)
  • ──► getHit (auto, se dispara desde TakeDamage)
